Spatial Discretization

The Challenge

Predicting continuous, infinite latitude/longitude coordinates is mathematically complex.

Discretizing Space

We converted the continuous map into discrete bounds using Uber's H3 Hexagonal Grid (Resolution 11 / ~20m diameter).

Creating a Neighborhood

For each Overture point, we generated a 19-hexagon neighborhood (K-Ring 2) and framed the problem as a classification challenge:

"Which of these 19 hexagons holds the TruePin?"

Feature Engineering

We queried the Overture Maps API for every hexagon to extract structural context:

  • Distance to Road Segments
  • Location of Building Footprints
  • Max Intersecting Building Area
